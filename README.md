# Folder Cleaner

A simple Python script that automatically organizes files in a folder by sorting them into subfolders based on file extensions.

## Features

- Automatic detection of file extensions  
- Creates folders per file type  
- Moves files into corresponding folders  
- Skips files if they already exist in the destination folder  
- No external libraries required  

## How to Run

Make sure Python is installed.

Set the folder path inside the script (line 34).

Then run:

```bash
python foldercleaner.py
```
